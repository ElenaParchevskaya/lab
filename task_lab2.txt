 No 2 Проектирование типов. Классы.

Задание
Определить класс типа Вектор. Реализовать методы для вычисления модуля вектор, скалярного произведения, сложения, вычитания, умножения на константу. 

Создать массив объектов, вывести:
А) векторы с заданным модулем,
Б) определить вектор с наибольшей/наименьшей суммой.


* не менее трех конструкторов с параметрами и без, + по умолчанию;
- статистический конструктор (конструктор типа);
- определить закрытый конструктор, варианты его вызова;
- поле только для чтения (например для каждого экземпляра поле для чтения ID  - равно некоторому уникальному номеру (хешу) вычисленному автоматически на основе инициализаторов объекта);
- поле - константу;
- свойства (get,set) - для всех поле класса (поля класса должны быть закрытыми). Для одного из свойств ограничьте доступ по set;
- в одном из методов класса для работы с аргументами используйте ref - и -out-параметры;
- создайте в классе статическое поле, хранящее количество созданных объектов (инкремируется в конструкторе) и статический методы выводы информации о классе;
- сделайте класс partical;
Переопределяйте методы класса Object: Equals, для сравнения объектов, стандартными рекомендациями, ToString - вывода строки - информации об объекте;

* Создайте несколько объектов вашего типа. Выполните вызов конструктора, свойств, методов, сравнения объектов, проверьте тип созданного объекта и тп;
